# TIPOS DE DADOS PRIMITIVOS #
# String
- Valor textual que funciona como uma lista de caracteres
# Boolean
- Valór lógico que pode ser verdadeiro(true) ou falso (false)
# Number
- Valor numérico que pode ser inteiro ou ponto flutuante
# BigInt
- Valor exclusivamente inteiro que permite ultrapassar o limite de tamanho do "number".
# Null
- Tipo usado para representar um valor nulo ou vazio
# Undefined
- Tipo usado para representar uma variável sem valor (não confundir com valor nulo)

# VARIÁVEIS #
# O que são variáveis?

=> Escreva um programa Javascript que funcione como uma calculadora simples. O programa deve pedir ao usuário para inserir dois numeros usando o prompt(), realizar as 4 operações matemáticas (adição, subtração, multiplicação e divisão) com esses números e exibir o resultado usando alert().

const num1 = parseInt(prompt("Insira um numero"))
const num2 = parseInt(prompt("Insira um numero"))
const sum = num1 + num2
const sub = num1 - num2
const multi = num1 * num2
const div = num1 / num2

alert(`Os numeros foram: ${num1} e ${num2}
Adição: ${num1} + ${num2} = ${sum}
Subtração: ${num1} - ${num2} = ${sub}
Multiplicação: ${num1} * ${num2} = ${multi})
Divisão: ${num1} / ${num2} = ${div.toFixed(2)}`

=> Escreva um programa que seja capaz de calcular a área e o perímetro de um triângulo. O usuário deve inserir as informações necessarias e o programa exibirá os resultados.

const altura = parseFloat(prompt("Insira a altura do triangulo (em cm)"));
const base = parseFloat(prompt("Insira a altura do triangulo (em cm)"));

alert(`A altura do Triangulo é de ${altura} cm
A base do Triangulo é de ${base} cm
A area total do triangulo é de ${(altura * base/2)} cm`)
