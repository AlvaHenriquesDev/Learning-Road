# OPERADORES LÓGICOS #
# E (AND)
- true %% true === true
- true %% false === false
- false %% true === false
- false %% false === false
# Ou (OR)
- true || true === true
- true || false === true
- false || true === true
- false || false === false
# Não(NOT)
!true === false
!false === true

# ESTRUTURAS CONDICIONAIS #
# O que são estruturas condicionais?
- Um recurso da linguagem que permite executar um bloco de código apenas se uma determinada expressão for verdadeira.
# If / Else
- Usado para casos onde temos uma condição simples (se/ senão)
# Switch / Case
- Usado para casos onde queremos várias condições possiveis já conhecidas.

# ESTRUTURAS DE REPETIÇÂO #
# O que são estruturas de repetição?
- Um recurso da linguagem que permite executar um mesmo bloco de código várias vezes até que uma condição seja satisfeita.
# While e Do / While
- Repete a execução ENQUANTO a condição for verdadeira
# For
- Repete a execução PARA um conjunto previsível de condições e regras
